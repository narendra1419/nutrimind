# NutriMind

NutriMind is an AI-powered personalized diet and fitness web application that helps users create customized meal plans, workout routines, and track fitness progress based on their health data.

## Features
- AI Diet Plan Generator
- Personalized Workout Plan
- BMI Calculator
- Progress Tracker
- AI Fitness Coach

## Tech Stack
Frontend: HTML, CSS, JavaScript  
Backend: ASP.NET Core (C#)  
Database: Supabase (PostgreSQL)

## Goal
The goal of NutriMind is to provide a smart and easy-to-use platform that helps users improve their health and maintain a balanced lifestyle using AI-powered recommendations.
